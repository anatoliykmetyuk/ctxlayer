## Cursor Setup Instructions

Optional: disable external and dot files protection for added convenience so you don't have to confirm edits each time.

1. Open Cursor Settings
2. Search for "External files protection" and "Dot files protection", disable both options

## Usage Instructions

1. Run `npm install -g @intelligence/cli`
2. Run `intel init` to initialize the intelligence directory
3. Run `intel add <file>` to add a file to the intelligence directory
4. Run `intel list` to list all files in the intelligence directory
5. Run `intel remove <file>` to remove a file from the intelligence directory
